Bem-vindo à Droptools 👋

A Droptools é uma plataforma criada para reunir algumas das ferramentas mais poderosas do mercado num único lugar, permitindo que utilizes tudo o que precisas de forma centralizada, simples e eficiente.

Em vez de pagares múltiplas subscrições separadas, alternares entre dezenas de sites ou gerires várias contas diferentes, a Droptools oferece uma solução unificada que facilita o teu dia-a-dia.

Este manual guia-te passo a passo na utilização da plataforma para que possas tirar o máximo partido desde o primeiro momento.

Checkout

Como fazer o Checkout

Após escolheres o plano mais adequado na Droptools, serás direcionado para a página de checkout.

  • Preenche todos os dados corretamente, sem erros.
  • Um preenchimento correto permite que o acesso seja ativado de forma automática e sem atrasos.
  • Após a confirmação do pagamento, poderás entrar em todas as salas da Droptools dentro do The Profit Blueprint.
1.1
Discord ID

Como obter o teu Discord ID

No checkout existe um campo DISCORD ID — é extremamente importante preenchê-lo corretamente, pois será através dele que receberás acesso às salas da Droptools.

  • Abre o Discord.
  • No canto inferior esquerdo, clica no teu nome de utilizador com o botão direito do rato.
  • Seleciona a opção "Copy User ID".
  • Cola esse ID no campo DISCORD ID durante o checkout.
⚠️ Atenção: Certifica-te de que o ID está correto antes de finalizar a compra para garantir que o acesso é atribuído sem atrasos.
2
A Tua Conta

Onde encontro os dados da minha conta?

Após o checkout, os dados de acesso serão enviados para o email que indicaste.

  • Abre o teu email e procura por mensagem de [email protected].
  • No email encontrarás: link para o Discord, username e password gerada automaticamente.
  • Guarda estes dados num local seguro.
⚠️ Não recebes o email? Verifica a pasta SPAM / Lixo eletrónico. Se mesmo assim não encontrares, confirma se o endereço introduzido no checkout estava correto.
3
Instalação Dicloak

Instalar o Dicloak

O Dicloak é o browser antidetect que permite acederes às ferramentas com as contas partilhadas.

  • Acede a www.dicloak.com e clica em Download.
  • Escolhe a versão correta para o teu sistema operativo: Windows ou macOS.
  • Instala o programa no teu computador.
  • Após a instalação, abre o Dicloak e inicia sessão com o username e password que recebeste no email.
4
Utilização

Como utilizar as ferramentas

Após o login no Dicloak, verás todas as aplicações disponíveis na dashboard.

  • Para cada aplicação existe um browser próprio associado.
  • Clica em "Open" para abrir a aplicação pretendida.
  • O browser abre automaticamente com o login já efetuado — sem necessidade de introduzir credenciais.
  • Poderão ser abertas 2 ou mais abas automaticamente — usa apenas a aba da aplicação que escolheste.

Códigos de verificação

Em algumas aplicações pode ser solicitado um código de confirmação.

  • Vai ao canal #droptools-codes no Discord.
  • Copia o último código disponível na sala.
  • Insere-o na aplicação para prosseguir.
⚠️ Usa sempre o código mais recente — códigos anteriores podem ter expirado.
